
Author: Minh Do

Function:
- Load image from image file (.jpg, .png ...)
- Display image on screen

Requirements:
- Linux OS
- OpenCV (tested with version 3.4.6)
- Cmake

How to run application:

- Open linux terminal: Ctr + Alt + T 
- cd 1.load-display-image
- run: ./run_app.sh
